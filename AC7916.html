Project Overview

AC7916AB 眼部按摩仪

demo_edr 是一款基于杰理 AC79NN 芯片的智能眼部按摩仪演示项目，采用分层架构设计， 集成蓝牙音乐、多模式按摩、LED 指示、复杂硬件驱动等功能。 项目代码约 3600 行，展示了 IoT 设备完整的软硬件一体化设计方案。

AC79NN 双核处理器 LLVM 编译 蓝牙 5.2 A2DP 4 种按摩模式 PWM 振动控制 气泵系统 温热按摩 SPP 数传
芯片: 杰理 AC79NN  ·  SDK: AC79NN_SDK_V1.2.0  ·  编译: LLVM  ·  Flash: 4 MB · SDRAM: 2 MB

1

2

工作模式

系统支持 5 种工作状态，通过单按键（PA1）短按快速循环切换：

模式LED 指示功能组合适用场景
MODE_SHUT_DOWN全灭全部关闭待机状态
MODE_COMFORTLED1（PA2）气泵 + 缓慢充放气舒适护理
MODE_HOTLED2（PA3）气泵 + 加热（50%）温热按摩
MODE_VIBRATIONLED3（PA4）气泵 + 振动（50% PWM）强效刺激
MODE_PASSIONLED4（PA5）气泵 + 加热 + 振动交替全功能组合

LED 控制

LED 为低电平点亮（推挽输出），引脚 PA2–PA5 分别对应 4 个工作模式：

// Inf_led.c - 低电平点亮，高电平熄灭
void Inf_led_turn_on(uint32_t led) {
    gpio_direction_output(led, 0);  // 输出 0V，LED 点亮
}

void Inf_led_turn_off(uint32_t led) {
    gpio_direction_output(led, 1);  // 输出 3.3V，LED 熄灭
}

舒适模式工作时序

模式执行周期为 1.8 秒（180 × 10 ms），分为四个阶段，实现自动充放气按摩效果：

1
缓慢放气（0–60 计数，0.6 秒） — 每 20 ms 周期性开关放气阀，防止残留气体伤害眼睛
2
快速放气（60–90 计数，0.3 秒） — 持续打开放气阀，彻底清空气囊
3
快速充气（90–120 计数，0.3 秒） — 关闭放气阀，打开充气阀，气囊快速膨胀
4
缓慢充气（120–180 计数，0.6 秒） — 周期性充气，微调压力到最舒适状态，循环重置
// App_mode.c: App_comfort_play() 核心逻辑（5ms 周期调用）
void App_comfort_play(void) {
    mode_task_count++;

    if (mode_task_count < 60) {          // 阶段1: 缓慢放气
        if (((mode_task_count - 30) % 20) < 10) {
            Inf_stop_input(); Inf_stop_output();
        } else {
            Inf_start_output();
        }
    } else if (mode_task_count < 90) {   // 阶段2: 快速放气
        Inf_start_output();
    } else if (mode_task_count < 120) {  // 阶段3: 快速充气
        Inf_stop_output(); Inf_start_input();
    } else if (mode_task_count < 180) {  // 阶段4: 缓慢充气
        if (((mode_task_count - 30) % 20) < 10) {
            Inf_stop_input();
        } else {
            Inf_start_input();
        }
    } else {
        mode_task_count = 0;             // 复位，开始下一周期
        Inf_stop_output(); Inf_stop_input();
    }
}

气泵系统

气泵系统由三路 GPIO 控制：气泵电机（PC4）、充气阀（PC5）、放气阀（PC6），HIGH 电平触发。

功能GPIO启动电平驱动对象
气泵电机PC4 (PUMP_PIN)HIGH气泵
充气阀PC5 (INPUT_PIN)HIGH充气电磁阀
放气阀PC6 (OUTPUT_PIN)HIGH放气电磁阀
// Inf_extern.h - 气泵引脚定义
#define PUMP_PIN   IO_PORTC_04   // 气泵电机
#define INPUT_PIN  IO_PORTC_05   // 充气阀门
#define OUTPUT_PIN IO_PORTC_06   // 放气阀门

// 控制函数
void Inf_start_PUMP(void)   { gpio_direction_output(PUMP_PIN,   1); }
void Inf_stop_PUMP(void)    { gpio_direction_output(PUMP_PIN,   0); }
void Inf_start_input(void)  { gpio_direction_output(INPUT_PIN,  1); }
void Inf_stop_input(void)   { gpio_direction_output(INPUT_PIN,  0); }
void Inf_start_output(void) { gpio_direction_output(OUTPUT_PIN, 1); }
void Inf_stop_output(void)  { gpio_direction_output(OUTPUT_PIN, 0); }
设计亮点：先清空残留气体防止二次伤害，快速膨胀产生按摩效果，缓慢充气精细调节压力，达到最佳体验。

A2DP 音乐播放流程

1
蓝牙接收 — 手机通过 A2DP 协议发送音频流
2
RTP 解析 — 提取音频包，解析 RTP 头部（序列号、时间戳）
3
音频解码 — 支持 SBC / AAC / APTX 多格式实时解码
4
混音处理 — 与本地提示音混音，应用音量控制（5–100，步长 5）
5
DAC 输出 — 输出到扬声器播放
