# BLE 功率计应用实现总结

## 📋 项目概述

本项目是一个基于 Flutter 开发的 BLE（蓝牙低功耗）设备监控应用，主要用于监控功率计设备的电流、电压、功率和耗电量数据。应用支持多种设备类型，包括功率计、码表和遥控设备。

## 🔧 技术架构

### 核心技术栈

- **框架**: Flutter 3.x
- **状态管理**: GetX
- **数据库**: SQLite (sqflite + sqflite_common_ffi)
- **蓝牙**: flutter_blue_plus
- **图表**: fl_chart
- **权限**: permission_handler
- **音频/震动**: audioplayers + vibration
- **网络**: dio (为 MQTT 和 OTA 预留)

### 架构模式

- **MVC 架构**: 清晰的控制器、模型、视图分离
- **服务层**: 数据库服务、报警服务、扫描设置服务
- **响应式编程**: 基于 GetX 的响应式状态管理

## 📊 数据流程

### 1. **数据采集流程**

```
BLE设备广播 → 扫描结果处理 → 数据解析 → 去重检查 → 阈值检查 → 数据库存储
```

### 2. **报警流程**

```
数据更新 → 阈值比较 → 报警触发 → 震动/声音 → 通知显示 → 冷却计时
```

### 3. **统计计算流程**

```
历史数据查询 → 时间分组 → 梯形积分计算 → 统计汇总 → 图表展示
```
