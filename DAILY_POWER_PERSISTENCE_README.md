# 每日耗电量统计数据持久化功能

## 🎯 功能概述

成功解决了耗电量统计数据丢失的问题，实现了完整的每日耗电量统计数据持久化功能。现在软件重启后，耗电量统计信息会被正确保存和恢复。

## ✅ 实现的功能

### 1. 数据库结构升级

- **数据库版本升级**: 从版本 2 升级到版本 3
- **新增数据表**: `daily_power_consumption` 表用于存储每日耗电量统计
- **表结构设计**:
  ```sql
  CREATE TABLE daily_power_consumption (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    deviceId TEXT NOT NULL,
    date INTEGER NOT NULL, -- 日期的毫秒时间戳（只包含年月日，时分秒为0）
    consumption REAL NOT NULL DEFAULT 0.0,
    dataPoints INTEGER NOT NULL DEFAULT 0,
    createdAt INTEGER NOT NULL,
    updatedAt INTEGER NOT NULL,
    FOREIGN KEY (deviceId) REFERENCES devices (deviceId),
    UNIQUE(deviceId, date)
  )
  ```

### 2. 数据持久化机制

- **自动保存**: 每小时自动保存已完成的每日耗电量统计数据
- **启动时加载**: 软件启动时自动从数据库加载已有的每日耗电量统计数据
- **停止时保存**: 停止监控时立即保存所有待保存的统计数据
- **智能去重**: 避免重复保存相同日期的数据

### 3. 核心技术实现

#### 新增数据模型

```dart
class DailyPowerConsumption {
  final String deviceId;
  final DateTime date; // 统计日期（只包含年月日，时分秒为0）
  final double consumption; // 当日耗电量 (mAh)
  final int dataPoints; // 当日数据点数量
}
```

#### 实时统计算法

- **梯形积分法**: 使用相邻两个数据点的平均电流乘以时间差计算耗电量增量
- **每日重置**: 新的一天自动重置统计，清空前一天的数据
- **跨天处理**: 正确处理跨日期边界的数据统计

#### 数据持久化流程

1. **数据接收**: 接收到新的设备数据时立即计算耗电量增量
2. **实时统计**: 实时更新内存中的每日耗电量统计
3. **定时保存**: 每小时自动保存已完成的每日统计到数据库
4. **启动恢复**: 软件启动时从数据库加载所有历史统计数据
5. **异常处理**: 网络异常或数据库错误时的优雅降级处理

## 🔧 技术细节

### 数据库操作方法

- `saveDailyPowerConsumption()` - 保存单条每日耗电量统计
- `getDailyPowerConsumptionObjects()` - 获取设备的每日耗电量统计对象
- `saveDailyPowerConsumptionBatch()` - 批量保存每日耗电量统计
- `deleteOldDailyPowerConsumption()` - 删除指定天数之前的旧数据

### 监控控制器集成

- **启动时加载**: `_loadDailyConsumptionStats()` 方法加载历史数据
- **运行时统计**: `_updateDailyConsumption()` 方法实时计算统计
- **定时保存**: `_startDailyStatsSaveTimer()` 启动定时保存任务
- **停止时保存**: `stopMonitoring()` 时保存所有待保存数据

### 数据完整性保证

- **原子操作**: 使用数据库事务确保数据一致性
- **唯一性约束**: 同一设备同一天只能有一条统计记录
- **索引优化**: 为查询频繁的字段建立索引
- **错误恢复**: 数据库操作失败时记录日志，不影响主功能

## 📊 数据流程示例

### 软件启动流程

1. 加载保存的设备列表
2. 为每个设备加载历史每日耗电量统计数据
3. 恢复内存中的每日耗电量统计状态
4. 继续实时统计新数据

### 数据统计流程

1. 接收新设备数据
2. 计算与前一数据的时间差和平均电流
3. 更新当前天的耗电量统计
4. 定时保存已完成的每日统计到数据库
5. 跨天时自动重置统计并保存前一天数据
